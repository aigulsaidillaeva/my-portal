# ИНСТРУКЦИЯ ПОЛЬЗОВАТЕЛЯ ПОРТАЛОМ

## 1. Вход в систему

*1.1 В адресной строке наберите https://bilim.gov.kg/integration/*  
*1.2 Перед Вами откроется главная страница сайта, на правой верхней стороне страницы нажмите кнопку ВХОД*  
*1.3 На странице «Аутентификации» выберите учреждение, логин и пароль*

## 2. Страница работы с факультетом

*2.1 Перед вами откроется личный кабинет со списком факультетов или с пустым списком*  
*2.2 Если нужно добавить факультет, нажмите кнопку*  
*2.3 После откроется модальное окно. Заполните форму*  
*2.4 Далее на главной странице увидите список факультетов*  
*2.5 Для редактирования нажмите кнопку. Для удаления — соответствующую кнопку*

## 3. Страница работы с направлением

*3.1 В панели управления выберите пункт “Направление”*  
*3.2 Перед вами откроется страница работы с направлениями*  
*3.3 Затем выберите факультет*  
*3.4.1 Если нет — перейдите в пункт 2.2*  
*3.4.2 Если есть — выберите факультет*  
*3.5 Для добавления направлений нажмите на кнопку*  
*3.6 После появится модальное окно. Нужно заполнить поля и сохранить*  
*3.7 Затем Вы увидите в списке направление*

## 4. Страница работы со специальностью

*4.1 В панели управления выберите пункт “Специальность”*  
*4.2 Перед вами откроется страница работы со специальностями*  
*4.3.1 Выберите факультет*  
*4.3.2 Если нет факультета — перейдите в пункт 2.2*  
*4.4.1 Выберите направление*  
*4.4.2 Если нет направления — перейдите в пункт 3.5*  
*4.5 Выберите форму обучения*  
*4.6 Для добавления специальности нажмите на кнопку*  
*4.7 После откроется модальное окно. Нужно заполнить и сохранить*

## 5. Страница работы со студентами

*5.1 В панели управления выберите пункт “Реестр учащихся”*  
*5.2 Далее откроется страница для работы со студентами*  
*5.3 Заполните все поля*  
*5.4 Для зачисления или восстановления нажмите на кнопку, затем выберите тип. После откроется модальное окно для добавления данных студента, которые нужно заполнить*  
*5.5 Далее на странице работы со студентами выйдет список студентов*  
*5.6 Для редактирования личных данных нажмите*  
*5.7 Для изменения (группы, курса и т.д.) нажмите, затем откроется страница студента. Затем выберите тип изменения:*  
  - *группа – изменение группы*  
  - *курс – перевод курса*  
  - *специальность – перевод по специальности*  
  - *академический отпуск – уход на академический отпуск*  
  - *отчисление – отчисление*  
  - *окончил – выпуск*  
  - *убыл – убыл по собственному желанию*  
*5.8 Далее откроется модальное окно, заполняем данные и сохраняем*
