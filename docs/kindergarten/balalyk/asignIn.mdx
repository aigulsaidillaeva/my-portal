## Общая информация

Данная страница является основной страницей авторизации пользователей в систему **"Балалык"**. Она предоставляет интерфейс входа для различных типов пользователей: ДОО (Дошкольные образовательные организации), Родители и Министерство/Финансисты. 👥


### 2. Навигация

- **Путь:** Главная - Вход 🏠 ➡️ 🚪 (информирует о текущем расположении пользователя в структуре сайта).

### 3. Переключатель ролей пользователя

Разрешает выбрать одну из следующих ролей:

- **ДОО** (по умолчанию активна) 🏫
- **Родители** 👪
- **Министерство** 🏛️ (полная надпись может быть: "Министерство образования" или другая).

### 4. Форма входа

- **Поле 1: ПИН**
  - Label: ПИН 🔢
  - Тип: Текстовое поле
  - Маска: 14 цифр (пример: `23112199901234`)
  - Иконка: Человечек 👤 (пользователь)
  
- **Поле 2: Пароль**
  - Label: Пароль 🔒
  - Тип: Пароль (скрытые символы)
  - Иконка: Замок 🔐

### 5. Ссылки под полями

- [Регистрация ДОО 📝](#) — ссылка на форму регистрации для дошкольных организаций.
- [Забыли пароль? 🔄](#) — ссылка для восстановления пароля.

### 6. Кнопка входа

- **Текст:** Войти ▶️
- **Цвет:** Зеленый 🟢
- **Действие:** Авторизация пользователя на основе введённых ПИН и пароля.

### 7. Альтернативные методы входа

Под формой есть надпись:

> Или войдите в систему с помощью 🔑

#### Варианты доступны в зависимости от выбранной роли пользователя:

| Роль        | Доступные методы входа      |
|-------------|------------------------------|
| ДОО         | Мугалим 🎓, ЕСИ / ТУНДУК 🛂 |
| Родители    | Только ЕСИ / ТУНДУК 🛂      |
| Министерство| Только Инфаком ID 🏢        |

## Особенности

- Система поддерживает разные типы пользователей. 🤝
- Интеграция с государственными сервисами (Тундук, ЕСИ, Инфаком). 🇰🇬
- Адаптивный и чистый интерфейс. 📱💻
- Безопасный ввод пароля. 🔐
- Автоматическое переключение альтернативных методов входа в зависимости от выбранной роли. 🔄

## Возможные сценарии использования

- ДОО входит для управления данными об учреждениях и детях. 🏫👶
- Родители авторизуются для подачи заявлений, просмотра информации о ребёнке и прочего. 👪📝
- Министерство и финансисты входят для анализа, отчетности, мониторинга заявок и распределения средств. 🏛️📊

## Примечания

- ПИН должен быть валидным и соответствовать формату (14 цифр). ✅
- Пароль скрыт по умолчанию. 🔒
- Возможна регистрация новых ДОО через ссылку. 📝
- Реализована система восстановления пароля. 🔄
